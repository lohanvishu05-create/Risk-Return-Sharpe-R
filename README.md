# Risk–Return and Sharpe Ratio Analysis using R

## Objective
To analyze investment portfolio performance using R by calculating returns, volatility, annualized risk, and Sharpe Ratio using Excel data.

## Tools Used
- R Programming
- Excel
- readxl package

## Analysis Performed
- Imported Excel data into R
- Calculated returns
- Measured volatility using standard deviation
- Annualized risk using √12 rule
- Computed Sharpe Ratio
- Visualized risk and performance

## Files
- Risk_Return_Sharpe_Analysis_R.R – R script
- returns_data.xlsx – Input data
- risk.png – Risk comparison chart
- sharpe.png – Sharpe Ratio chart

## Key Insight
Sharpe Ratio helps identify the most efficient investment by balancing return and risk.
